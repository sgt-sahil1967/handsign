Help & Frequently Asked Questions

Getting Started

How do I start using the application?

  1. Navigate to the Detection page
  2. Click the "Start Webcam" button
  3. Allow camera permissions when prompted by your browser
  4. Show your hand to the camera
  5. View the predictions in real-time

What browsers are supported?

The application works best with modern browsers that support:

  • Chrome 60+ (Recommended)
  • Firefox 55+
  • Safari 11+
  • Edge 79+

Note: Internet Explorer is not supported.

Camera Issues

The webcam won't start. What should I do?

  • Check permissions: Make sure you've allowed camera access in your browser settings
  • Close other apps: Other applications (Zoom, Teams, etc.) might be using the camera
  • Check browser console: Press F12 and look for error messages
  • Try a different browser: Some browsers handle permissions differently
  • Restart browser: Sometimes a simple restart fixes permission issues

Camera is blurry or not detecting my hand

  • Ensure good lighting - avoid backlighting
  • Keep your hand at a comfortable distance (30-60cm from camera)
  • Make sure your entire hand is visible in the frame
  • Avoid cluttered backgrounds
  • Clean your camera lens

Detection Issues

No predictions are showing

  • Make sure your hand is clearly visible in the camera frame
  • Try different hand positions and angles
  • Ensure adequate lighting
  • Wait a few seconds - predictions update every 500ms
  • Check the browser console (F12) for errors

Predictions are inaccurate

  • Make sure you're showing one of the supported hand signs clearly
  • Keep your hand steady for better recognition
  • Ensure your hand is fully in the frame
  • Try adjusting the distance from the camera
  • Check that lighting is adequate

What hand signs are supported?

The application recognizes 8 hand signs:

  • Open (open palm)
  • Close (closed fist)
  • Pointer (pointing with index finger)
  • OK (thumb and index finger forming a circle)
  • ASL A (fist with thumb on side)
  • ASL B (flat hand, fingers together)
  • ASL C (curved hand, like letter C)
  • ASL D (index finger pointing up, other fingers closed)

Technical Issues

The page won't load

  • Make sure the Flask server is running (python app.py)
  • Check that you're accessing http://localhost:5000
  • Check the terminal for error messages
  • Try refreshing the page (Ctrl+R or Cmd+R)

I see "Connection refused" error

The Flask server is not running. Start it with:

cd "/path/to/project"
source venv311/bin/activate
python app.py

Predictions are slow

  • This is normal - predictions are processed every 500ms
  • Processing happens on the server, which adds network latency
  • For faster predictions, ensure good internet connection (if applicable)
  • Close other applications to free up system resources

Privacy & Security

Is my video data stored or sent anywhere?

No. All processing happens locally:

  • Video is captured in your browser
  • Frames are sent to your local Flask server
  • No data is sent to external servers
  • No video is stored or saved

Can I use this offline?

Yes, once the server is running, you can use the application offline. However, you need:

  • The Flask server running locally
  • All model files present in the project directory
  • No external dependencies required

Performance

What are the system requirements?

  • CPU: Any modern processor
  • RAM: 2GB minimum, 4GB recommended
  • Camera: Any webcam or built-in camera
  • Browser: Modern browser with WebRTC support

Why is my computer slow when using this?

  • Machine learning inference is computationally intensive
  • Close other applications to free up resources
  • Ensure your computer meets the minimum requirements
  • Consider using a more powerful device if available

Still Need Help?

If you're still experiencing issues:

  • Check the browser console (F12) for error messages
  • Check the Flask server terminal for backend errors
  • Review the How It Works page for technical details
  • Ensure all dependencies are properly installed
